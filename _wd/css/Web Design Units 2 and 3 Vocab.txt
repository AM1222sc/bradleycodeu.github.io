Hypertext Markup Language -        the language for describing the proper formatting of text and images so that your Internet browser may display them as they are intended to look


hypertext -        a word, phrase or chunk of text that contains a link to another file or webpage


tag -        an HTML keyword written inside of triangle brackets. They usually are paired with a closing version that consists of: opening triangle bracket, a forward slash, the same HTML keyword, and closing triangle bracket


head -        contains important info about an HTML page, but is not displayed on the HTML page. It is placed between the html tag and the body tag. Typically contains the document title, character set, styles, scripts, and other meta information


body -        contains all the contents displayed within an HTML page, such as text, hyperlinks, images, tables, lists, etc.


title -        defines a name for the browser tab and provides a name for the page when it is added to favorites/bookmarks


h1 -        defines the most important headings in an HTML page


p -        defines a paragraph in an HTML page. Browsers will automatically give it a top margin and bottom margin.


br -        inserts a single line break in an HTML page. It is an empty tag which means that it has no closing tag


bold -        text that is thicker, darker, and heavier than normal


italic -        text that is slightly slanted in order to give it emphasis


img -        defines an image in an HTML page. It has two required attributes: src and alt


alt -        provides alternative information for an image if a user cannot view it (because of an error in loading the image, slow connection, or if the user uses a screen reader)


ol -        defines a numerically or alphabetically sorted list in an HTML page


ul -        defines a bulleted, un-numbered list in an HTML page


li -        defines a list item in an HTML page. It must be placed within ol or ul tags


Cascading style sheets -        the language for describing the presentation of Web pages, including colors, layout, and fonts


Inline CSS -        used to define a style for a single HTML tag. For example, h1 style="color:blue;"


Internal CSS -        used to define a style for a single HTML page, this code is placed within style tags inside of the head section


External CSS -        used to define the style for one or more HTML pages. It allows you to change the look of an entire web site by changing one file.


CSS rule -        a statement that tells browsers how to render a specific HTML tag. It consists of a selector, curly bracket, one or more declarations, and a closing curly bracket.


CSS selector -        the name of the HTML tag that will be styled by the following declarations.


CSS declaration -        a CSS property paired with a value. For example, color: blue;


Hexadecimal -        a base-16 number system that uses the values 0-9, A, B, C, D, E and F


CSS box model -        a set of CSS properties that wrap around every HTML element. It consists of: margins, borders, padding, and the content


Padding -        a CSS property that generates space around an element’s content within the border


Border -        a CSS property that creates a line around the padding and the content


Margin -        a CSS property that creates space around an element, outside of the border


Font Stack -        CSS code that tells a browser what typeface to display and what backups to use if the first choice is not available


Serif Font -        a typeface that contains small extra lines at the ends of characters. For example, Times New Roman


Sans Serif Font -        a typeface that does not use small lines at the ends of characters. For example, Arial and Helvetica


table -        an HTML element for creating rows and columns on a Web page


tr -        defines a row in an HTML table and contains one or more th or td elements


th -         a header cell in an HTML table


td -        a normal data cell in an HTML table


255, 255, 255 -        RGB values for white


255, 255, 0 -        RGB values for yellow


0, 255, 255 -        RGB values for cyan/aqua


255, 0, 255 -        RGB values for magenta/fuchsia


0, 0, 0 -        RGB values for black


#FFFFFF -        Hexcode for white


#FF0000 -        Hexcode for red


#FFFF00 -        Hexcode for yellow


#00FF00 -        Hexcode for green


#00FFFF -        Hexcode for cyan/aqua


#0000FF -        Hexcode for blue


#FF00FF -        Hexcode for magenta/fuchsia


#000000 -        Hexcode for black
